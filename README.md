# Nexus — Responsive Navigation Bar 🧭

> A sleek, accessible, mobile-first navigation bar built with pure HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://responsive-nevigation-bar.vercel.app/
💻 **GitHub Repo:** https://github.com/pavanpatil06459/Responsive-Nevigation-Bar

---

## ✨ Features

- 🍔 Animated hamburger menu with smooth transitions
- 🔒 Sticky glassmorphism navbar with backdrop blur
- 📱 Fully responsive mobile-first layout
- ♿ Accessibility support using ARIA attributes
- ⌨️ Keyboard friendly (Escape key closes menu)
- 🖱️ Outside-click closes mobile menu
- 🔗 Active link sync for desktop and mobile nav
- 🌀 Scroll-aware navbar border effect
- 🎨 Easy theming using CSS variables

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6+)  
- Responsive Design  
- Accessibility (ARIA)
